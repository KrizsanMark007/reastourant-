# Restaurant Website

## Overview
This project is a simple restaurant website that showcases the menu, provides information about the restaurant, and includes a contact section for inquiries. The website is designed to be user-friendly and visually appealing.

## Project Structure
```
restaurant-website
├── index.html          # Main HTML document for the restaurant website
├── assets
│   ├── css
│   │   └── styles.css  # CSS styles for the website
│   └── js
│       └── main.js     # JavaScript code for interactive elements
└── README.md           # Documentation for the project
```

## Setup Instructions
1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd restaurant-website
   ```
3. Open `index.html` in your web browser to view the website.

## Features
- Menu section displaying various dishes
- About section with information about the restaurant
- Contact form for customer inquiries

## Technologies Used
- HTML
- CSS
- JavaScript
